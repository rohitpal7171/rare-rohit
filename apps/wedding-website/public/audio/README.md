# public/audio/ — Ceremony Music Files

Place MP3 files here. The CeremonyMusicPlayer component
looks for these exact filenames. Missing files are silently
ignored — the player hides itself if the file returns a 404.

## Required files

| File          | Ceremony | Suggested mood                    |
|---------------|----------|-----------------------------------|
| haldi.mp3     | Haldi    | Folk dhol, folk songs             |
| mehendi.mp3   | Mehendi  | Soft ghazal, acoustic folk        |
| sangeet.mp3   | Sangeet  | Upbeat Bollywood, celebratory     |
| baraat.mp3    | Baraat   | Shehnai + dhol, procession beats  |
| pheras.mp3    | Pheras   | Vedic mantras, shehnai, classical |
| vidaai.mp3    | Vidaai   | Emotional classical, soft shehnai |

## Sources for royalty-free music
- https://pixabay.com  (search "indian wedding", "dhol", "shehnai")
- https://freesound.org
- https://www.jamendo.com

## File size
Keep each file under 3MB. Use 128kbps MP3.
